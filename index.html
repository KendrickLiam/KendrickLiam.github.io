Encryption & HTTPS

Encryption is the process of scrambling data so that only the intended recipient can read it. When you visit a website that uses HTTPS, your browser and the server perform a “handshake” that securely exchanges encryption keys. After that, all information you send or receive is protected by those keys, meaning that even if someone intercepts the data, it looks like unreadable code. HTTPS also ensures authenticity by verifying the website’s identity with digital certificates, so you know you’re communicating with the right site.
